yum安装redis

阿里云Centos7 yum安装redis

1.首先安装epel源

yum install epel-release

2.安装redis

yum install redis

3.安装完毕后，使用下面的命令启动redis服务

# 启动redis
service redis start
# 停止redis
service redis stop
# 查看redis运行状态
service redis status
# 查看redis进程
ps -ef | grep redis

4.设置redis为开机自动启动

chkconfig redis on

5.redis-cli进入redis服务

# 进入本机redis
redis-cli
# 列出所有key
keys *

6.打开配置文件，修改密码，IP等

vi /etc/redis.conf

①可以修改端口（默认为6379）

②可以修改密码

 ③可以修改允许远程登录

1.默认只允许本机登录

bind 127.0.0.1

2.可以指定IP登录（指定ip，多个ip之间使用空格分隔）

bind 127.0.0.1 192.168.3.3

3.允许所有IP登录（不安全）

bind 0.0.0.0

7.修改完配置文件后，可以通过如下命令使配置文件生效

redis-server /etc/redis.conf &

或者

# 停止redis
service redis stop
# 启动redis
service redis start

8.远程登录redis

(其中127.0.0.1可以更改为远程公网IP)

redis-cli -h 127.0.0.1 -p 6379
